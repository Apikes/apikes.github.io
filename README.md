# apikes.github.io

### Interface

|                          |                              |

| -------------------------| -----------------------------|

| **Ctrl + +/-**           | Scale interface              |

| **Ctrl + N**             | New note                     |

| **Ctrl + E**             | Toggle edit/preview mode     |

| **Alt + Left** (custom)  | Navigate back                |

| **Alt + Right** (custom) | Navigate forward             |
